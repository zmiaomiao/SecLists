<p align="center"> 
<font color="#800080"><b>如何找到你想找的网页</b></font></p>
<p align="left"> 
感谢您访问 LiDaBoJi 立达博济培训网。如果您直接在地址栏输入一个您访问过的网页的地址，别忘了有的网址可能有大小写。例如，本站使用的是Unix系统，而部分网页文件名
正是用大写字母的。例如：</p>
<p align="center"> 
论语：<a href="http://www.ldbj.com/LunYu.htm">http://www.ldbj.com/<font color="#FF0000">L</font>un<font color="#FF0000">Y</font>u.htm</a></p>
<p align="center"> 
英语：<a href="http://www.ldbj.com/English.htm">http://www.ldbj.com/<font color="#FF0000">E</font>nglish.htm</a></p>
<p align="center"> 
<a href="../ZhangZhiyong.htm">张智勇</a> <a href="../Zhang-Zhiyong.htm">我的培训之路</a></p>
<p align="left"> 
您有几种方法找到您想找的网页：</p>
<p align="left"> 
第一： <a href="../map.htm">培训网站内导航</a></p>
<p align="left"> 
站内导航就是一个网站地图，网站的大的结构、主要的内容分类排列在这里。</p>
		<p align="left"> 
通过“心理学 <a href="../psychology/psychology.htm">
		Psychology</a>”栏目可以看到：<a href="../psychology/mengdejiexi.htm">梦的解析</a>，但是<a href="../psychology/zhougongjiemeng.htm">周公解梦</a>并不在导航栏目出现，虽然可以通过“网页索引”和“相关阅读”找到。</p>
<p align="left"> 
第二：<a href="../wangyesuoyin/wangyesuoyin.htm">网页索引</a>
<a href="../wangyesuoyin/wangyesuoyin_a.htm">A</a> 
<a href="../wangyesuoyin/wangyesuoyin_b.htm">B</a>
	<a href="../wangyesuoyin/wangyesuoyin_c.htm">C</a> 
<a href="../wangyesuoyin/wangyesuoyin_d.htm">D</a>
	<a href="../wangyesuoyin/wangyesuoyin_e.htm">E</a> 
<a href="../wangyesuoyin/wangyesuoyin_f.htm">F</a>
	<a href="../wangyesuoyin/wangyesuoyin_g.htm">G</a> 
<a href="../wangyesuoyin/wangyesuoyin_h.htm">H</a>
	<a href="../wangyesuoyin/wangyesuoyin_i.htm">I</a> 
<a href="../wangyesuoyin/wangyesuoyin_j.htm">J</a>
	<a href="../wangyesuoyin/wangyesuoyin_k.htm">K</a> 
<a href="../wangyesuoyin/wangyesuoyin_l.htm">L</a>
	<a href="../wangyesuoyin/wangyesuoyin_m.htm">M</a> 
<a href="../wangyesuoyin/wangyesuoyin_n.htm">N</a>
	<a href="../wangyesuoyin/wangyesuoyin_o.htm">O</a> 
<a href="../wangyesuoyin/wangyesuoyin_p.htm">P</a>
	<a href="../wangyesuoyin/wangyesuoyin_q.htm">Q</a> 
<a href="../wangyesuoyin/wangyesuoyin_r.htm">R</a>
	<a href="../wangyesuoyin/wangyesuoyin_s.htm">S</a> 
<a href="../wangyesuoyin/wangyesuoyin_t.htm">T</a>
	<a href="../wangyesuoyin/wangyesuoyin_u.htm">U</a> 
<a href="../wangyesuoyin/wangyesuoyin_v.htm">V</a>
	<a href="../wangyesuoyin/wangyesuoyin_w.htm">W</a> 
<a href="../wangyesuoyin/wangyesuoyin_x.htm">X</a>
	<a href="../wangyesuoyin/wangyesuoyin_y.htm">Y</a> 
<a href="../wangyesuoyin/wangyesuoyin_z.htm">Z</a></p>
<p align="left"> 
网页索引把网站主要内容按照26个字母的顺序排列。如果您要找《<a href="../guwenguanzhi/guwenguanzhi.htm">古文观止</a>》，在“<a href="../wangyesuoyin/wangyesuoyin_g.htm">网页索引字母G</a>”下面可以找到。</p>
<p align="left"> 
举例说明：</p>
<p align="left"> 
《古文观止》的例子不好，不能说明在字母“G”下面是照“古”字排列，还是照“观”字排列。</p>
		<p align="left"> 
<a href="../yangpijuan/zuiweidadetuixiaoyuan.htm">世界上最伟大的推销员</a>的关键词采用的是“最”，可以在字母“Z”下查到，但是因为“推销”实际上是该书的核心，所以也可以在字母“T”下找到。</p>
		<p align="left"> 
当然也可以在字母“Y”下找到，因该书亦名<a href="../yangpijuan.htm">羊皮卷</a>。</p>
<p align="left"> 
第三：相关阅读</p>
<p align="left"> 
相关的内容有时候被放在每个网页最后的相关阅读栏目里，有时候直接放在网页中间。</p>
		<p align="left"> 
所以如果您找到了<a href="../libai/libai.htm">李白</a>您就能够找到 <a href="../ffff/1.htm">杜甫</a> 
<font face="宋体"><a href="http://www.neixun.cn/xinggan/115.htm">白居易</a></font>             
<a href="../liqingzhao/liqingzhao.htm">李清照</a>                              
 <a href="../tupian/10.htm">柳永</a> <a href="../luyou/luyou.htm">陆游</a>。</p>
		<p align="left"> 
如果您找到了<a href="../zuidigongzi/zuidigongzi_beijing_2010.htm">北京市2010年最低工资标准</a>，要找到<a href="../zonghegongshi/zonghegongshi_2005_94.htm">北京市企业实行综合计算工时工作制和不定时工作制行政许可实施规定</a>就很容易了。</p>
		<p align="left"> 
同样如果您找到了<a href="../zhengce_fagui/jihuashengyu.htm">中共中央、国务院关于进一步做好计划生育工作的指示</a>，那么通过“相关阅读”的链接-有时候在上面，有时候在正文，有时候在下面，您很快就能找到有关产假、陪产假的规定。</p>
