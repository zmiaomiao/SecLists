	    <h2>About Us</h2>
		<hr/>
	      
	AlliedModders is an open-source development community.  We are focused on server-side modifications to any and all games on the Half-Life 1 and Half-Life 2 platforms.  Our primary projects are:
	</p>
	<ul>
	 <li>
	 <p>
	  <img src="/media/images/Sm_small.gif" alt="SourceMod Logo" /> <a href="http://www.sourcemod.net/">http://www.sourcemod.net/</a>
	  <br />
	  SourceMod is an extensible, actively maintained server administration and scripting project for Half-Life 2.  Its goals are clean API, good documentation, and high performance.</p>
	 </li>
	 <li>
	  <p>
	  <img src="/media/images/Amxxsmall.gif" alt="AMX Mod X Logo" /> <a href="http://www.amxmodx.org/">http://www.amxmodx.org/</a>
	  <br />
	  AMX Mod X is an extensible server administration and scripting project for Half-Life 1.  It is now maintained on a need-basis only.</p>
	 </li>
	 <li>
	  <p>
	  <img src="/media/images/Mms.jpg" alt="Metamod:Source Logo" /> <a href="http://www.metamodsource.net/">http://www.metamodsource.net/</a>
	  <br />
	  Metamod:Source is a powerful C++ plugin loader and API for Half-Life 2.  It wraps a highly versatile library for detouring virtual functions.
	  </p>
	 </li>
	 <li>
	  <p>
	  <img src="/media/logos/amb2.png" alt="AMBuild Logo" /> <a href="https://wiki.alliedmods.net/AMBuild">https://wiki.alliedmods.net/AMBuild</a>
	  <br />
	  AMBuild is a fast, accurate build system geared toward C++ projects.
	  </p>
	 </li>
	</ul>
