In the background, in more detail -  the database has been improved. By using a performance-optimized MySQL software, we were able to achieve a performance gain of over 600%. This is a secure and sustainable growth, the page load time has become in the folder listing even faster.

In the folder overview, we have included the number of saved links for each cyberlocker. So you have a better overview how many links of one cyberlocker are in one folder. Mirrors and folders that are offline for two months will be deleted after two months. This applies to all time Share.cx, Bitload.com und Duckload.com mirror / folder.
