		<br><br>
Партнёрская сеть Actionpay является одним из ведущих агрегаторов партнёрских программ в интернете, работающих по CPA-модели с оплатой за действие. Модель CPA (cost-per-action) или PPA (pay-per-action) является наиболее эффективной моделью размещения рекламы в интернете, так как рекламодатель оплачивает не просто рекламные показы или переходы на свой сайт, а достижение конкретных целей, которые напрямую связаны с получением прибыли.
<br><br>
Партнёрская сеть Actionpay в настоящий момент предлагает для работы вебмастеров множество партнёрских программ. Основная география офферов - Россия, Украина, Казахстан, Молдова, а также в сети представлены офферы для международного трафика. CPA-(PPA-)программы с оплатой за действие представлены по множеству тематик: банки и финансы, интернет-магазины, купоны и скидки, красота и здоровье, инфо-товары, туризм и отдых, авиабилеты, развлечения и многое другое. Почти все партнёрские программы принимают все "белые" типы трафика.
<br><br>
Если вы рекламодатель, то с помощью Actionpay вы сможете качественно повысить эффективность своих сайтов, значительно оптимизировать свои расходы на рекламу в интернете. Ваша партнёрская программа в нашей сети получит максимально таргетированный поток посетителей, при этом вы не оплачиваете посещения, а платите только за достижение целей - заполнение заявок, регистрации, покупки, заказы, оставленные контактные данные, скачивание цифрового контента, например игр или приложений.
<br><br>
Если вы вебмастер, то пройдя за одну минуту процесс регистрации, вы можете стать нашим партнёром, и, получив доступ в свой личный кабинет и выбрав подходящие вашему трафику или аудитории ваших сайтов офферы из представленного каталога партнёрских программ, вы можете начать зарабатывать прямо сейчас. Для вас еженедельные выплаты, низкие холды, круглосуточная служба технической поддержки и при значительном трафике персональный менеджер, который будет держать вас в курсе новостей и предложит индивидуальные условия сотрудничества, а также один из самых крупных каталогов партнёрских программ.
<br><br>
С партнёрской сетью Actionpay зарабатывают все!

            if (typeof data == 'function') {

					}
					self.loaded();

					if (self.autoheight) {

            }

				self.fitHeight();

            window.signinLoadingDeffered = this.$el.find('#popup-signin-loading').show();

            return this;
        },

            window.signinLoadingDeffered = this.$el.find('#popup-signin-loading').fadeOut();

            return this;
        },

        render: function (data) {
            this.content.html('')
                .append(data)
            ;

            return this;
        },

		fitHeight: function (animate) {
			var maxHeight = parseInt('80%');
			if ('80%'.indexOf('%') > -1) {
				maxHeight = 0.01 * maxHeight * $(window).height();
			}
			var realHeight = 0;
			this.content.children().filter(':visible').each(function () {
				realHeight += $(this).outerHeight(true);
			});
			if ('') {
				realHeight += 60;
			}
			if (realHeight > maxHeight) {
				realHeight = maxHeight;
			}

			var contentHeight = realHeight;
			var frontHeight = contentHeight + 40 + 10;
			if (animate) {
				this.content.animate({height: contentHeight});
				this.$front.animate({height: frontHeight});
			} else {
				this.content.height(contentHeight);
				this.$front.height(frontHeight);
			}
		},

        hide: function () {
			            this.$el.fadeOut('fast');

			window.signin.onHide();
        },

<script>
	$(function () {
		var popupShown = false;
		if (window.signin) {
			window.signin.onShow = function () {
				popupShown = true;
			}
		}
		window.initSignin = function (callback) {
			if (!window.hasOwnProperty('grecaptcha')) {
				setTimeout(function () { window.initSignin(callback) }, 100);
				return;
			}
			send(null, callback);
		};
		var $form = $('#auth-form');
		var currentStep;
		var redirectAfterSignin = "";
		var sending = false;
		var $allControls = $('input, select, button', $form);

		function send(data, callback) {
			$allControls.attr('disabled', true);
			if (data) {
				if (!Auth.checkEcDone || !Auth.checkFpDone) {
					setTimeout(function () {
						send(data, callback);
					}, 200);
					return;
				}

				data.timezone = new Date().getTimezoneOffset();
				data.fonts 	  = sha1(''+JSON.stringify(Auth.fontList));
				data.plugins  = Auth.plugins();
			}
			if (sending) return;
			sending = true;
			$.ajax({
				url: '/ru/session/',
				data: data,
				method: data ? 'post' : 'get'
			}).then(function (res) {
				if (currentStep != res.session.step) {
					currentStep = res.session.step;
					$('[data-step]', $form).css('display', 'none');
					var $step = $('[data-step="' + currentStep + '"]', $form).css('display', 'block');
					$step.find('input[type=text]').first().focus();
				}
				if (res.captcha) {
					$('.g-recaptcha', $form).show();
					grecaptcha.reset();
				} else {
					$('.g-recaptcha', $form).hide();
				}
				if (res.error) {
					$('.alert', $form).show().find('span').text(res.error);
				} else {
					$('.alert', $form).hide();
				}

				$('.auth-login', $form).text(res.session.login || '');
				$('.auth-username', $form).text(res.session.username || '');
				$('[name=code]', $form).val('');
				if (res.session.subusers) { var $subusers = $('#auth-form-subusers-list', $form);
					$('li:not([data-template])', $subusers).remove();
					res.session.subusers.forEach(function (subuser) {
						var $item = $('li[data-template]', $subusers).clone().removeAttr('data-template');
						for (var field in subuser) if (subuser.hasOwnProperty(field)) {
							$('[data-bind-text="' + field + '"]', $item).text(subuser[field]);
							$('[data-bind-value="' + field + '"]', $item).attr('value', subuser[field]);
						}
						$item.appendTo($subusers);
					});

					$('#auth-form-subusers-list label', $form).click(function () {
						var id = $(this).attr('value');
						$(this).closest('form').find('[name=subuser]').val(id);
						$(this).closest('form').submit();
					});
				}

				if (res.session.totp) {
					$('#auth-form-use-totp', $form).show();
				} else {
					$('#auth-form-use-totp', $form).hide();
				}

				if (res.session.login && !res.session.authorized) {
					$('#auth-form-logout', $form).show();
				} else {
					$('#auth-form-logout', $form).hide();
				}

				sending = false;
				if (popupShown) {
					window.signin.fitHeight(true);
				}

				if (res.session.step == 'completed') {
					setTimeout(function () {
						window.location.replace(redirectAfterSignin);
					}, 1000);
				}
				if (typeof callback === 'function') {
					callback();
				}

				$allControls.removeAttr('disabled');
			})
		}


		$('[data-step]', $form).submit(function () {
			var data = {};
			$(this).serializeArray().forEach(function (field) {
				data[field.name] = field.value;
			});
			send(data);
			return false;
		});

		$('#auth-form-logout a', $form).click(function () {
			$.get(this.href).then(function () { send() });
			return false;
		});

		$('#auth-form-use-sms a', $form).click(function () {
			send({use_sms: '1'});
			return false;
		});
		$('#auth-form-use-totp a', $form).click(function () {
			send({use_totp: '1'});
			return false;
		});

		$('[name=code]', $form).on('keyup mouseup change', function () {
			var prvlen = $(this).data('prevlength')| 0;
			var curlen = $(this).val().length;
			var maxlen = $(this).attr('maxlength') | 0;
			$(this).data('prevlength', curlen);
			if (curlen > prvlen && curlen >= maxlen) {
				$(this).closest('form').submit();
			}
		});
	})
</script>
