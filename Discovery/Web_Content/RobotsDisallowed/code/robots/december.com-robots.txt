<h3>Common Errors</h3>
<p>A common error is to attempt to access a URL on december.com with a ".htm" filename extension.  The december.com site uses all ".html" filename extensions. For example, this URL will not work: 
<blockquote class="code"> http://www.december.com/cmc/mag/1998/jul/baker.htm</blockquote>
But this URL will work:
<blockquote class="code"><a href="http://www.december.com/cmc/mag/1998/jul/baker.html">http://www.december.com/cmc/mag/1998/jul/baker.html</a></blockquote>
