<h2>認証に失敗しました</h2>
<p>UMIN 401 Unauthorized</p>



<h3>以下の原因が考えられますのでご確認願います。</h3>


<h4>ご利用のUMIN ID、パスワードが間違っている可能性がございます</h4>
<ul>
	<li >
		【ユーザ名＝UMIN ID】、【パスワード＝UMIN IDのパスワード】となります。
	</li>
	<li >
		UMIN ID、パスワードは半角英数字となります。
	</li>
	<li >
		UMIN ID、パスワードの前後に半角スペース、全角スペースは入れないでください。
	</li>
	
	<li >
		UMIN IDの入力に「@umin.ac.jp」や「@umin.net」「@umin.org」は必要ございません。
	</li>
	<li >
		小文字、大文字、数字の入力ミスにご注意ください。
	</li>
	<li>
		パスワード更新申請を行った場合、完了通知が来ていることを確認してください。完了通知から約1時間お待ち頂く必要がございます。
	</li>
</ul>

<h4>制限のあるページでアクセス権限が無い可能性があります</h4>
<ul>
	<li >
		まず、ご利用のUMIN IDとパスワードが正しいことを【<a href="https://center6.umin.ac.jp/newuser.htm">ログインテスト</a>】でご確認ください。
		<dl>
			<dt>「本ページが表示されたら、正しいIDとパスワードが入力されたことを示します。」</dt>
			<dd>UMIN IDとパスワードが正しいため、アクセス権限が無いようです。ご利用のページの管理者様にお問い合わせください。</dd>
			<dt>「認証に失敗しました」のページが表示される。</dt>
			<dd>UMIN IDまたは、パスワードに誤りがあります。</dd>
		</dl>
	</li>
</ul>


<h3>UMIN IDまたはパスワードを失念してしまった方へ</h3>
