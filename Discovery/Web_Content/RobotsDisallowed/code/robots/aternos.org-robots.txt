Aternos is not a 24/7 host. Your server is only running as long as you need it and play on it. When you leave your server, it will be stopped automatically. But your server is saved and you can start it whenever you want to.

No. Aternos is contrary to all other free minecraft server hosts free for everyone. We want that every user is equal and has the same opportunities.
