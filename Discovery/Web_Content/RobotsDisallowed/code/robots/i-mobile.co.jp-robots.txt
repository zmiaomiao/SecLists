<h1>Web サイトが見つかりません</h1>
アクセスしようとする Web サイトの IP アドレスが、ポート番号を指定する要求を受け付けないように構成されています。
<hr>
<p>対処方法</p>
<ul>
<li>ブラウザのアドレス バーに表示された Web サイト アドレスの綴りと形式が正しいことを確認してください。</li>
<li>リンクをクリックしてこのページに到達した場合は、Web サイトの管理者に連絡して、リンクの形式が正しくないことを通知してください。
</li>
<li>別のリンクへアクセスするには、[<a href="javascript:history.back(1)">戻る</a>] をクリックしてください。</li>
</ul>
<h2>HTTP エラー 404.1 - ファイルまたはディレクトリが見つかりません : 要求されたポートで Web サイトにアクセスできません。<br>インターネット インフォメーション サービス (IIS)</h2>
<hr>
<p>技術情報 (サポート担当者用)</p>
<ul>
<li>「<a href="http://go.microsoft.com/fwlink/?linkid=8180">Microsoft 製品サポート サービス</a>」で、&quot;HTTP&quot; および &quot;404&quot; の文字列によりタイトルを検索してください。</li>
<li>IIS マネージャ (inetmgr) からアクセスできる IIS の [ヘルプ] を開き、「単一サーバー上の複数の Web サイトをホストする」、「インターネット サービスへのポート割り当て」、および「カスタム エラー メッセージについて」というトピックを検索してください。</li>
</ul>
